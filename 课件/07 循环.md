## 循环

#### 1. for

- 基本介绍

```js
for( 语句1 , 语句2 , 语句4 ){
	//语句3 循环体代码
    //code
    //code
}

//执行顺序： 1  ->  2?  -> 3  -> 4 -> 2? -> 3 -> 4 -> 2 ……
```

#### 2. while

- while 和 do while

```js
//while
while( 条件 ){
	//循环体代码
}


//dowhile
do{
    //循环体代码
}while( 条件 )
```
